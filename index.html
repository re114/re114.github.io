Robot Draws

Teaching robots to draw.

This site is an index to ongoing research into machine learning and drawing. The overarching aim is to capture an image and extract the essence as a vector image capable of being drawn by a robot with a pen.

Jupyter notebooks and Colab

The text and code have been created as Jupyter Notebooks so the code samples can be run inline with the text, either by being downloaded to the reader's machine, or by opening the notebooks in Colab. There is an 'open in colab button at the head of notebooks with code. Click the button, select a runtime (GPU) and execute the code.
